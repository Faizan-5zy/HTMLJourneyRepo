* HTML Input Types Overview

This HTML file showcases various input types commonly used in web forms. Here's a brief overview of each input type:

1. **Text:**
   ```html
   <input type="text">
   ```
   - Standard single-line text input.

2. **Submit Button:**
   ```html
   <input type="submit">
   ```
   - Creates a button to submit a form.

3. **Colour:**
   ```html
   <input type="color">
   ```
   - Allows users to pick a color.

4. **Date:**
   ```html
   <input type="date">
   ```
   - Provides a date picker.

5. **Email:**
   ```html
   <input type="email">
   ```
   - Ensures the input is a valid email address.

6. **Hidden:**
   ```html
   <input type="hidden" value="Delhi">
   ```
   - Hidden input with a predefined value.

7. **Number:**
   ```html
   <input type="number">
   ```
   - Accepts numerical input.

8. **Password:**
   ```html
   <input type="password">
   ```
   - Conceals text input for sensitive information.

9. **Reset:**
   ```html
   <input type="reset">
   ```
   - Resets the form to its initial state.

10. **Time:**
    ```html
    <input type="time">
    ```
    - Allows users to pick a time.

11. **URL:**
    ```html
    <input type="url">
    ```
    - Ensures the input is a valid URL.

12. **Range:**
    ```html
    <input type="range" min="1200" max="2000">
    ```
    - Allows users to select a value from a specified range.

13. **Checkbox:**
    ```html
    <input type="checkbox">
    ```
    - Provides a checkbox for binary choices.

14. **Radio:**
    ```html
    <input type="radio">
    ```
    - Presents a radio button for mutually exclusive options.

15. **File:**
    ```html
    <input type="file">
    ```
    - Enables users to upload files.
